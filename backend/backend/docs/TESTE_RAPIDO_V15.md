# Teste rápido v15 — Plano de Contas Gerencial

## Objetivo
Validar que o plano de contas gerencial:
- já vem estruturado automaticamente para empresas novas
- pode ser editado, complementado e excluído na interface
- é usado nos módulos de Contas a Pagar, Contas a Receber e Tesouraria

## Fluxo sugerido
1. Entrar no sistema.
2. Abrir **Financeiro > Plano de Contas**.
3. Confirmar se a estrutura padrão foi carregada automaticamente.
4. Editar uma conta existente.
5. Criar uma nova subconta.
6. Excluir uma conta sem vínculos.
7. Abrir **Contas a Pagar**, **Contas a Receber** e **Tesouraria** e verificar se as contas aparecem nos seletores.
