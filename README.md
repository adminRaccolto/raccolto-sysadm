# Raccolto API v4

API NestJS do Raccolto.

## Principais evoluções desta versão
- clientes ampliados
- produtos e serviços administráveis
- contratos ampliados
- recebíveis automáticos básicos
- tarefas com atribuição flexível e base para lista/kanban no front

## Como rodar
```bash
cp .env.example .env
npm install
npm run prisma:generate
npm run prisma:push
npm run start:dev
```
