# Raccolto Web v2

Painel web do Raccolto para testar o backend no navegador.

## Principais evoluções desta versão
- clientes ampliados
- produtos e serviços
- contratos ampliados
- tarefas com lista e kanban
- tela inicial de financeiro

## Como rodar
```bash
cp .env.example .env
npm install
npm run dev
```
